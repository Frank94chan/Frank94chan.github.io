Arch Linux install tutorial

这是我自己在 Asus 笔记本上安装 Arch Linux 的过程记录。参考资料来源于 Arch 官方 Wiki

安装步骤

准备材料

  • 可用的网络
  • Arch Linux 最新版的 ISO 一份
  • 至少4 GB 的 U盘一个
  • Windows 下安装 Rufus 作为制作 U盘启动盘的工具，Linux 下用 dd 命令即可。
  • 支持 UEFI 启动的笔记本一个

第一步：制作启动盘

从官方网站，也可以是从国内各大镜像站，例如清华开源镜像站下载最新版的镜像，应选择最新版的镜像下载。下载完毕后，推荐验证一下文件的完整性，Download 列表最下方应该提供了 md5sums.txt 和 sha256sums.txt 这两个文件，Windows 下 7-zip 可以提供文件校验，Linux 下自带 md5sum 和 sha256sum 这两个命令。

文件校验完成之后，Windows 用户打开我刚说的 Rufus 选择要写入镜像的 U盘和镜像，分区方案选择 UEFI 和 GPT分区，确认这三个选项没有问题之后可以点击 开始，等待完成即可。

到这里，供 UEFI 电脑启动的启动盘就做好了。

第二步：进入启动盘并安装基本系统

重启电脑，进入 Bios，选择从 U盘启动。没有问题的话，你现在应该看到一个黑色的背景上有几个选项，进入第一个即可。闪过几页系统启动日志，Live CD 会自动以 root 身份登录，从现在开始，你将亲手打造自己的 Arch Linux。

系统分区并挂载

我的硬盘组合是120GB SSD + 500GB HDD，我给自己的分区方案是：

  • SSD 500MB EFI 分区 /boot
  • 剩下的作为根分区 /
  • HDD 6GB 做 Swap 分区
  • 剩下的做家分区 /home
